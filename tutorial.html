การใช้งานที่ทั้งสอง editor ทำได้เหมือนกัน
มาเรียนรู้ความหมายของปุ่มใน emacs กันก่อน!

Modifier keys เป็นปุ่มที่จะต้องใช้ร่วมกับปุ่มอีก ได้แก่ Alt, Ctrl และ Shift รวมไปถึง
Meta, Super และ Hyper ในการใช้ emacs ปุ่มที่เราจะสนใจหลัก ๆ คือ Ctrl และ Meta
ในการใช้ปุ่ม Ctrl จำเป็นจะต้องกดค้างไว้ (holding) และกดปุ่มอื่นที่ต้องการ
แต่ในการใช้ปุ่ม Meta นั้น ไม่จำเป็นที่จะต้องกดปุ่ม Meta ค้างไว้ สามารถละมือไปกดปุ่มอื่นได้เลย
(ปุ่ม Meta ในที่นี่จะหมายถึง ESC หรือ Alt)

  • C หมายถึง ปุ่ม Ctrl

  • M หมายถึง Meta key ซึ่งหมายถึง Alt หรือ ESC

สำหรับผู้ใช้ Mac

ใน Mac ซึ่งไม่มีปุ่ม Alt ให้ใช้งานจึงต้องใช้งานปุ่ม ESC แต่หากต้องการใช้ปุ่ม option แทน
ก็สามารถทำได้ โดยเข้าไปตั้งค่าตามนี้

    Terminal > Preferences > Settings > *Whatever profile you use* > Keyboard

และติ้กในช่อง

  • "Use option as meta key"

Modesการใช้งาน ของ vim

Modes ใน vim เป็นการแบ่งการทำงานผ่าน keyboard เป็นหลาย ๆ โหมดด้วยกัน
โดยแบ่งออกเป็นโหมดต่าง ๆ ดังนี้ (ในที่นี่ขอไม่กล่าวถึงอีก 3 โหมดได้แก่ select, command-line และ ex-mode)

    1. Normal

      1.1 การที่เข้าโหมด normal ให้กดปุุ่ม ESC

      1.2 ใช้ในการนำทางและการจัดการข้อความ (text)

      1.3 การกดตัวอักษรบนแป้มพิมพ์จะเป็นการใช้คีย์ลัดในการทำงาน

    2. Insert

      2.1 กด i เพื่อเข้าสู่โหมด insert

      2.2 การกดตัวอักษรบนแป้มพิมพ์จะเป็นการเพิ่มตัวอักษรตัวนั้นลงไปในไฟล์

      2.3 การจะใช้คีย์ลัดไม่สามารถใช้ได้ตรง ๆ จะต้องใช้ Ctrl หรือ Alt เป็นต้น

      2.4 โดยปกติ emacs และ text editor อื่น ๆ จะอยู่ในโหมด insert เป็นหลัก

    3. Visual

      3.1 กด v เพื่อเข้าสู่โหมด visual

      3.2 ใช้ในการเลือกขอบเขตของ text ที่เราต้องการ (ภาษาตลาดเรียกว่า คลุมดำ)

      3.3 หลังจากเลือก (selection) ได้ตามที่ต้องการแล้วใช้คีย์ลัดได้แบบใน normal mode โดยไม่ต้องกด ESC

  • โดยเริ่มต้นแล้ว vim จะเข้าอยู่ใน normal mode ให้

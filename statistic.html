What text editor do you use? (2011) by UtkarshSengar.com

       เมื่อ 5 ปีที่แล้ว Utkarsh Sengar ซึ่งเป็นผู้พัฒนาโปรแกรมคนหนึ่งบนโลกออนไลน์ได้ทำแบบสำรวจเกี่ยวกับ text editor ที่ผู้พัฒนาใช้และสรุปผลออกมาพบว่าผู้ใช้งาน vim มีมากถึง 48% และ emacs มีผู้ใช้งาน 21.4% ซึ่งยังน้อยกว่า vim อยู่ถึง 26.6% โดยจำนวนผู้เข้าร่วมทำแบบสำรวจนี้คือ 3,232 คน



Arch Linux Statistics (2016)

       Arch Linux เป็นอีกหนึ่ง Linux distribution ได้แสดงผลทางสถิติไว้ภายในเว็บไซต์ archlinux.de ซึ่งสถิตินี้เกี่ยวกับ editors ที่ผู้ใช้มีการติดตั้งและทำการใช้มาจากการที่ในผู้ใช้จะมีการติดตั้ง package ที่ชื่อว่า pkgstats ซึ่งถือว่าเป็น kernel modules ที่ทำงานอัตโนมัติในทุก ๆ สัปดาห์โดยจะส่งข้อมูลไปยังผู้พัฒนาของ Arch โดยส่งแบบไม่ระบุตัวตน (anonymous) และไม่สามารถระบุตัวตนของ user ที่ส่งข้อมูลมาได้

       จากสถิติของภาพด้านล่างนี้จะสังเกตได้ว่าผู้ใช้ยังมีความต้องการในการติดตั้งและใช้งาน vi และ vim มากคือ 86.86% และ 66.84% ตามลำดับ ในส่วนของ emacs นั้นอยู่ที่ 21.68% เท่านั้น ซึ่งจะสังเกตได้ว่า vi หรือ vim ยังเป็นที่ต้องการและเป็นที่ใช้งานของผู้ใช้งาน Arch Linux อยู่อย่างชัดเจน



What Editor Do Pythonists Use? (2015)

       เว็บไซต์ sitepoint.com ซึ่งเป็นเว็บที่สอนเกี่ยวกับเรื่องการทำเว็บไซต์ได้มีการสำรวจโดยการพูดคุยกับผู้ที่เขียนโปรแกรมภาษา Python 100 คน และพบว่า editor ที่ผู้พัฒนาโปรแกรม Python เลือกใช้เป็นอันดับ 1 นั้นคือ Sublime Text และอันดับที่ 2 ตามมาด้วย vim และอันดับที่ 3 คือ emacs ซึ่งก็แสดงถึงการที่ vim และ emacs ยังมีบทบาทในการช่วยผู้พัฒนาโปรแกรมอยู่ตั้งแต่อดีตจนมาถึงปัจจุบัน



Text Editor of Developer Survey (2015) by StackOverflow.com

       ผลการสำรวจนี้มาจากการสำรวจทั้งหมด 16,778 คนจากผู้พัฒนาทั่วโลกที่ถูกจัดขึ้นโดยเว็บไซต์ Stackoverflow.com พบว่า มีผู้พัฒนา (developer) ที่ใช้งาน vim 15.2% ซึ่งเยอะกว่า emacs ที่มีผู้พัฒนาใช้งานเพียง 3.8% โดยผลสำรวจนี้ถูกจัดทำขึ้นในปี 2015 ซึ่ง vim ก็ยังเป็น text editor ที่มีผู้พัฒนาใช้งานมากกว่า emacs อยู่ถึงแม้จะผ่านมาถึง 4 ปี แต่ถึงอย่างไรนั้น vim ก็ยังไม่ใช่ editor ที่ผู้พัฒนาที่มีผู้พัฒนาใช้งานมากที่สุดอยู่ดี สำหรับตำแหน่งของ text editor ที่มีผู้พัฒนาใช้งานมากที่สุดนั้นกลับเป็นของ Notepad++ และ Sublime Text แต่ในที่นี้เราจะสนใจเพียงผลสำรวจของ vim และ emacs เป็นสำคัญ

       ผลสำรวจนี้ค่อนข้างมีความน่าเชื่อถือมาก อันเนื่องมาจากเป็นการเปิดแบบสำรวจให้ผู้ใช้งานของเว็บไซต์ stackoverflow.com เข้ามาทำแบบสอบถามด้วยความสมัครใจและด้วยตัวเอง ดังนั้นผู้ใช้ที่เข้ามานั้นเป็นส่วนในกลุ่มของผู้ใช้ที่ active คือใช้งานอยู่ตลอด

       ยิ่งไปกว่านั้นเมื่อแบ่งออกเป็นผลสำรวจที่ขึ้นกับตำแหน่งการทำงานพบว่า ผู้ที่เป็น DevOps & Sys admins นั้นใช้งาน vim เป็น text editor มากที่สุดอีกด้วย



เกร็ดความรู้

       DevOps เกิดจากการรวมกันของคำว่า Development และ Operation รวมไปถึง QA (quality assurance) โดยได้แนวคิดมาจาก Agile และ Lean นำมารวมเข้าด้วยกัน โดย DevOps เป็นสิ่งที่จะช่วยลดความขัดแย้งและเพิ่มศักยภาพในการทำงานร่วมกันในองค์กรซึ่งจะทำให้การ development และ delivery software ดีขึ้น

       Sysadmins หรือชื่อเต็มคือ system administrator เป็นผู้ที่มีหน้าที่ดูแล ปรับแต่ง และควบคุมเกี่ยวกับระบบคอมพิวเตอร์ ยกตัวอย่างเช่น เซิร์ฟเวอร์ เป็นต้น
